ğŸ” HOW THE SYSTEM WORKS:

MASK DETECTION PIPELINE:
1. Capture frame from camera
2. Convert BGR to grayscale
3. Apply Haar Cascade for face detection
4. Extract face ROI (Region of Interest)
5. Preprocess (resize to 128x128, normalize)
6. Feed to CNN model
7. Get prediction (5 classes)
8. Calculate virus risk percentage
9. Draw colored bounding box
10. Display results

CROWD MONITORING PIPELINE:
1. Capture frame from camera or video
2. Run YOLOv11 inference
3. Filter detections for person class
4. Count total persons in frame
5. Compare with threshold
6. If threshold exceeded:
   - Trigger audio alert
   - Display warning on frame
   - Record alert event
7. Draw bounding boxes for all persons
8. Display counter on frame
9. Update statistics
